# HRMS Lite

HRMS Lite is a Full Stack Employee and Attendance Management System built using **React, Django REST Framework, and MySQL**.

This project allows administrators to manage employees and track their attendance efficiently.

---

# Tech Stack

Frontend
React (Vite)

Backend
Django REST Framework

Database
MySQL (Railway)

Deployment

Frontend → Vercel,
Backend → Render,
Database → Railway

---

# Features

Employee Management

* Add Employee
* Delete Employee
* View Employee List

Attendance Management

* Mark Attendance
* View Attendance History

---

# Project Architecture

React Frontend
↓
Django REST API
↓
MySQL Database

Frontend Setup

cd frontend

npm install

npm run dev
